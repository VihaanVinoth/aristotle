# How to visit project? 
I might host the demo soon, but for now, you can ```git clone``` in these steps:

```
git clone https://github.com/VihaanVinoth/aristotle.git
cd aristotle
mv aristotle.css [YOUR/FILE/PATH] (or simply drag into your folder)
```
...in your ```index.html```:

```
<link rel="stylesheet" href="aristotle.css">
```

...and you're done!

## Extras
If you want, you can also link your ```style.css```, but in reference to ```aristotle.css```, type ```!important``` next to the CSS property, like this:

```
color: red !important;
```
